1. Learnings are filed as GitHub issues under milestone **M-COMPETE** with labels `program:competitive-learn` + `status:backlog-only` + `type:learn`.
2. Implementation only after removing `status:backlog-only` and attaching to an implementation milestone (same SDD rule as gap backlog).
3. Prefer P0 themes that fit metapi-go’s operator meta-gateway shape (TTFT routing, trace correlation, channel auto-disable, cost breakdown).
